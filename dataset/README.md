# Dataset

This folder contains sample data used for development and testing of the parser module.

- **dataset.pdf**: A text-based university ledger PDF used by the application to validate parsing accuracy. Developers can replace this file with their own examples to experiment with different formats.

> Note: The parser expects text-based PDFs (not scanned images) following the structure described in the main README under "PDF Requirements."
